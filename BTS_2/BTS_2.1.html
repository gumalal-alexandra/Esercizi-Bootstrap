1. Qual è la dimensione del testo di default di bootstrap?

Bootstrap 5 usa un testo di default di: 1rem (16px)

2. Qual è il font di default di bootstrap?

Il font di default di bootstrap è system-ui, apple-system, Segoe UI, Roboto, Helvetica Neue, Arial, sans-serif

3. Quali sono il margine superiore e inferiore di un paragrafo?

Il margine superiore di un paragrafo è: margin-top: 0. Invece il margine inferiore è: margin-bottom: 1rem (16px)
