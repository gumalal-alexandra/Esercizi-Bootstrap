1. A cosa serve Bootstrap?

È un framework front-end che serve per creare siti web responsive e mobile-first, con componenti già pronti in HTML, CSS e JavaScript.

2. Cosa devi importare per creare una pagina con Bootstrap

Si deve collegare un CDN

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/css/bootstrap.min.css" rel="stylesheet">

3. Cos’è un CDN? Quali vantaggi ha?

È una rete di server che distribuisce file da località diverse nel mondo, per rendere il caricamento dei contenuti più veloce. Usare un CDN evita di scaricare Bootstrap sul tuo sito e migliora le prestazioni.

4. A cosa serve content="width=device-width, initial-scale=1"

Serve a dire al browser di adattare la larghezza del sito alla larghezza dello schermo del dispositivo, mantenendo la scala iniziale a 1. È necessario per il layout responsive.

5. Differenza tra container e container-fluid

container ha larghezze massime fisse e resta centrato. container-fluid invece si estende sempre al 100% della larghezza dello schermo.
